Edición estándar de Web "Integrantes Grupo 1"

Bienvenido a nuestro proyecto Web, una página completamente funcional que puede utilizar como descripción visual de los integrantes que conforman la cátedra de TIO en la carrera TUDAI de la Facultad de Ciencias Exactas.

Este documento contiene información sobre los requisitos de sistema y navegadores necesarios para poder acceder a nuestra Web. Para obtener una explicación más detallada, consulte el capítulo Requisitos de Navegación.

Requisitos de Navegación:

Usar Windows 10 a superior (recomendado)

Utilizar navegador Google Chrome o Mozilla Firefox

Podrá acceder a nuestro Repositorio público desde el siguiente enlace:
https://github.com/agustingroh/grupo1-tp-TIO

¿Qué hay en nuestra Web?

En la Web "Integrantes Grupo 1" proyecto que nace a partir de la necesidad de poder plasmar en una Web, nuestros nombres en una lista y demostrar la utilización del lenguaje de etiquetas HTML5, junto a JavaScript y CSS3 para mejor aspecto visual.

En ella encontraremos:

- Título
- Subtítulo
- Un cuerpo con una tabla y en ella el nombre de cada uno de los integrantes del Grupo 1
- Datos de contacto
-Un footer con referencias y condiciones legales


Todos los contenidos que se desarrollan en Web Integrantes Grupo 1, fueron lanzados bajo la licencia MIT o BSD.
